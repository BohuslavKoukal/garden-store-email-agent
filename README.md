# рҹҢҝ Garden Store Email Agent

An intelligent email-processing agent built with **LangChain** and **LangGraph** that automatically reads, classifies, and responds to customer emails for an online garden store.

## вңЁ Features

- рҹ“§ **Reads** all `.txt` email files from the `emails/` folder
- рҹҸ·пёҸ **Classifies** each email into `plant_instructions` or `other` intent using an LLM
- рҹҢҝ **Answers** plant-related questions with expert advice generated by GPT-4o
- вң… **Human-in-the-loop** approval before any answer is saved
- рҹ’ҫ **Stores** approved answers to the `answers/` folder
- рҹӨ– **Auto-replies** to non-plant emails with a polite personalised message

## рҹҡҖ Quick Start

### 1. Clone & install

```bash
git clone https://github.com/BohuslavKoukal/garden-store-email-agent.git
cd garden-store-email-agent

python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate

pip install -e ".[dev]"
```

### 2. Configure environment

```bash
cp .env.example .env
# Open .env and add your OPENAI_API_KEY
```

### 3. Add emails

Drop `.txt` files into the `emails/` folder. See `emails/example_plant.txt` and `emails/example_other.txt` for examples.

### 4. Run

**CLI:**
```bash
python -m ui.cli
```

**Web UI (Streamlit):**
```bash
streamlit run ui/streamlit_app.py
```
